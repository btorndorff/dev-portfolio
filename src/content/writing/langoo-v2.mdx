Langoo is a personalized language learning tutor that remembers your language learning journey and adapts to your needs to help you learn a language faster.

## Features

- **Human-like understanding and speaking** - Thanks to OpenAI's Speech-to-Speech model, Langoo can understand and respond naturally
- **Persisted Memory** - Langoo remembers his lessons with you thanks to Mem0
- **Tailored curriculums** - Personalized learning paths for each user based on their goals and learning style that updates as you progress

## Tech Stack

Built with React and Tanstack Query on the frontend, Hono.js with Drizzle ORM and SQLite on the backend. Uses OpenAI's realtime speech-to-speech API and Gemini API for the AI features. Deployed on Cloudflare with BetterAuth for authentication.
